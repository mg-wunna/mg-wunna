## Project purpose

A premium conversion-focused studio website designed to attract freelance clients вЂ” **not** a developer portfolio. Every section is built to move a visitor from attention to a qualified lead.

## Key features

- **Framer Porty-style work showcase** вЂ” every project is a cinematic case study, not a card in a grid. Full-bleed cover, editorial story sections, alternating gallery, next-project navigation.
- **Lead-qualifying contact form** вЂ” multi-step intake captures project type, budget, timeline, goals, and description. Stored to MongoDB via `POST /apis/leads`.
- **Apple / Linear / Stripe-inspired minimalism** вЂ” Inter, generous whitespace, subtle motion, one accent color.
- **Light + dark mode** вЂ” full parity via `next-themes` with a selector-based dark mode.
- **Framer Motion scroll reveals** вЂ” respects `prefers-reduced-motion`.

- Apple-style minimalism
- Studio-grade presentation (cinematic case studies, not card grids)

## Tech

- **Framer Motion** for restrained scroll reveals

- **Zod** input validation
